### Method 2: Automated Fix (Windows)

Create this batch file in your local project root and run it:

**File: `fix-android-build.bat`**
```batch
@echo off
echo Fixing Android build configuration...

mkdir "android\capacitor-cordova-android-plugins\src\main\libs" 2>nul

echo Creating cordova.variables.gradle...
(
echo // DO NOT EDIT THIS FILE! IT IS GENERATED EACH TIME "capacitor update" IS RUN
echo ext {
echo   cdvMinSdkVersion = project.hasProperty('minSdkVersion'^) ? rootProject.ext.minSdkVersion : 23
echo   // Plugin gradle extensions can append to this to have code run at the end.
echo   cdvPluginPostBuildExtras = []
echo   cordovaConfig = [:]
echo }
) > "android\capacitor-cordova-android-plugins\cordova.variables.gradle"

echo Running Capacitor sync...
npx cap sync android

echo Android build configuration fixed!
```
